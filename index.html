Washington Bike Demand Prediction

This project aims to predict the demand for bike rentals in Washington using machine learning and deep learning techniques to help companies like Zagster and Lime Bikes optimize bike availability across locations and time. By analyzing patterns in bike usage and applying models such as Linear Regression, Gradient Boosting, Decision Trees, KNN, Deep Neural Networks, and LSTM, the project addresses the challenge of supply-demand mismatch, ultimately aiming to reduce customer dissatisfaction and operational inefficiencies. Evaluation metrics like Mean Squared Error and Mean Absolute Error were used to assess model performance.

April 2025

Twitter Sentiment Analysis

This project focuses on predicting user sentiment (positive, negative, or neutral) from text data such as tweets and comments using natural language processing (NLP) and deep learning. By converting text into vectorized representations using Count Vectorizer and TF-IDF, and applying a deep neural network, the model aims to help companies analyze public sentiment during product launches. Exploratory Data Analysis revealed a predominance of neutral sentiments and highlighted frequently used words in both positive and negative contexts through word clouds. Despite signs of overfitting, hyperparameter tuning was conducted to optimize performance, enabling the model to provide actionable insights from user feedback to support data-driven business decisions.

April 2025

Predicting readability of texts using NLP

This project focuses on automating the prediction of text difficulty using machine learning and deep learning techniques, particularly for aiding readers and librarians in classifying vast amounts of published content. By leveraging NLP and vectorization methods like TFIDF Word2Vec, along with models such as deep neural networks and gradient boosted trees, the system effectively predicts readability scores. Exploratory data analysis revealed key features like sentence count and word length impacting difficulty. The best-performing model, a deep neural network with TFIDF Word2Vec encoding, showed strong predictive accuracy and potential for real-time integration into tools like Microsoft Word to assist users in gauging text complexity.

June 2025

RAG based medical chatbot

This is a web-based conversational AI application built using Streamlit. Its core functionality is a Retrieval-Augmented Generation (RAG) system, powered by Langchain. This system answers user queries by first retrieving relevant information from a FAISS vector store, and then uses a HuggingFace Endpoint with the HuggingFaceH4/zephyr-7b-beta model to generate coherent responses.
